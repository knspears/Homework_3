In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

I displayed nine different projections in QGIS, inclduing local projections (such as NAD 83 / North Carolina, which centers around North Carolina) and global projections (such as WGS84, which centers around the equator). I did this by changing the EPSG/ESRI projection in QGIS, exporting the map to the image editor (still in QGIS), and then adding the PNG image to Github and Visual Studio Code.
